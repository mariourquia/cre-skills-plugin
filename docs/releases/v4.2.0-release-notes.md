# v4.2.0 — Hardening pass 2 close (residential_multifamily beta RC)

## Summary

Pass-2 hardening closes the three objectives deferred from pass 1: sealed-close gating (Obj 5), finance-critical placeholder scanner (Obj 6), and executive output integrity (Obj 8). The `residential_multifamily` subsystem moves from `draft` to `beta_rc` (v0.6.0). 436 tests passing (+13 net new).

## Highlights

### Sealed-close gating (Obj 5)
- `_core/schemas/period_seal.yaml` canonicalizes close_status ordering and version fields.
- `_core/final_marked_workflows.yaml` adds `period_grade_workflows` registry (6 slugs).
- Six workflow manifests declare `required_period_seal` aligned with the registry.
- `tests/test_period_seal_gating.py` (5 tests) enforces the contract.

### Finance placeholder scanner (Obj 6)
- `_core/reference_data_integrity.md` defines the placeholder-labeling rule.
- `tests/test_finance_placeholder_scanner.py` (4 tests) scans every CSV read by a final-marked workflow manifest and rejects unlabeled placeholders.

### Executive output contract (Obj 8)
- `_core/executive_output_contract.md` defines verdict-first structure, five source-class labels, and refusal-artifact shape.
- Four final-marked SKILL.md files reference the contract; the canonical example demonstrates the full pattern.
- `tests/test_executive_output_contract.py` (4 tests) enforces references and example shape.

## Migration notes

- Reference manifests may now declare an optional `required_period_seal` top-level field. Schema validation accepts manifests without it (backward compatible).
- Period-grade workflows (monthly/quarterly/executive summary, reforecast, budget build) will REFUSE when close_status is below the declared floor. Declared floors are conservative (`soft_close` for operating, `hard_close` for board/LP/lender).
- Executive output is now required to carry source-class tags on numeric cells. Existing authors should consult `_core/executive_output_contract.md`.

## Gates and validation

- Changed files: 24 (manifests, schemas, SKILL.md, examples, 3 new tests).
