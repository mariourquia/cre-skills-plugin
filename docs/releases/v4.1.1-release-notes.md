# v4.1.1 — Release asset parity

## Summary
Adds desktop and portable build targets, source archives, cosign signing, and consolidated checksums to the release pipeline. Brings release artifacts to 4-target parity (macOS DMG, Windows EXE, desktop zip, portable zip).

## Highlights
- Desktop and portable zip targets added to `release.yml`
- Source archives attached to each release
- Cosign signing for all release artifacts
- Consolidated `SHA256SUMS` file for cross-platform verification
- Install docs updated to describe each artifact and verification flow

## Gates and validation
- Release workflow now produces 4 platform targets per tag
- Checksums published with each release asset

See `docs/INSTALL.md` for per-target verification.
