# v4.1.0 — Marketplace support, installer hardening

## Summary
Same-repo marketplace support, Cowork install guidance, structured installer telemetry, and CI-validated plugin.json placement. This release unblocked `/doctor` by moving `plugin.json` to the repo root and hardened installers with edge-case handling and structured telemetry.

## Highlights
- Same-repo marketplace install path: `claude plugin marketplace add <repo>`
- Cowork install guidance added (Desktop zip upload flow)
- Installer telemetry: structured events for install start, success, failure, and edge cases
- PS1 installer: step-by-step progress and error reporting
- Windows installer verification fixed; plugin cache layout corrected

## Gates and validation
- Added CI validation for marketplace metadata
- `/doctor` now recognizes the plugin at repo root

## Install
See `docs/INSTALL.md` and `docs/install-cowork.md`.

## Upgrade notes
Users on v4.0.x should reinstall via the marketplace path; legacy manual installs remain supported but are no longer recommended.
