## What's New in v2.0.0

### Orchestrator Integration
This release integrates the [CRE Acquisition Orchestrator](https://github.com/ahacker-1/cre-acquisition-orchestrator) by Avi Hacker -- a multi-agent pipeline engine for CRE acquisitions. The plugin's skills now serve as execution methodologies for the orchestrator's specialist agents, adding institutional-grade depth to every pipeline phase.

### 8 New Skills (13,701 lines + 24 reference files)

| Skill | Lines | Refs | What It Does |
|-------|-------|------|-------------|
| `title-commitment-reviewer` | 766 | 3 | Title analysis with state-specific practices (25+ states), exception categorization, cure paths |
| `tenant-credit-analyzer` | 626 | 3 | HHI concentration, WALT-weighted credit scoring, default probability modeling, worked examples |
| `term-sheet-builder` | 650 | 3 | Term sheet drafting across 6 loan types, rate lock strategy, negotiation playbook |
| `loan-document-reviewer` | 928 | 3 | Covenant testing against projections, carve-out analysis, intercreditor review |
| `transfer-document-preparer` | 680 | 3 | Entity authorization, 1031 exchange coordination, FIRPTA compliance, transfer tax (50 states) |
| `funds-flow-calculator` | 790 | 3 | Prorations (3 methods), settlement statements, wire fraud prevention, worked $15M example |
| `lease-option-structurer` | 646 | 3 | Option package design, collision detection, termination fee valuation, 3-alternative comparison |
| `lease-trade-out-analyzer` | 529 | 3 | Renewal vs trade-out NPV comparison, breakeven analysis, risk-adjusted recommendation |

### 9 Python Calculators (zero dependencies)

Executable scripts that agents can run for precise quantitative output:

| Script | Skill | Calculations |
|--------|-------|-------------|
| `proration_calculator.py` | funds-flow | Per diem prorations (actual/365, 30/360) |
| `npv_trade_out.py` | lease-trade-out | NPV comparison, breakeven, 2D sensitivity grid |
| `tenant_credit_scorer.py` | tenant-credit | HHI, WALT, expected annual loss, OCR |
| `covenant_tester.py` | loan-doc-reviewer | DSCR/LTV/debt yield by year, breach detection |
| `option_valuation.py` | lease-option | Termination fees, cap rate impact, package NPV |
| `waterfall_calculator.py` | jv-waterfall | GP/LP distributions, multi-tier promote, IRR |
| `debt_sizing.py` | loan-sizing | DSCR/LTV/DY constraint optimization |
| `transfer_tax.py` | transfer-doc | All 50 states + DC with tiered rates |
| `quick_screen.py` | deal-quick-screen | Cap rate, DSCR, CoC, replacement ratio, 3-scenario IRR |

### Brand Guidelines System

Skills that produce investor-facing deliverables (pitch decks, IC memos, quarterly updates) now auto-load your brand guidelines. Set up once with `/cre-skills:brand-config`, and all future deliverables use your colors, fonts, disclaimers, and contact info automatically.

**8 deliverable skills updated:** ic-memo-generator, lp-pitch-deck-builder, quarterly-investor-update, capital-raise-machine, fund-formation-toolkit, disposition-prep-kit, investor-lifecycle-manager, leasing-strategy-marketing-planner

### Telemetry & Feedback (Opt-In Only)

Anonymous, local-only usage tracking to help improve the plugin. **Disabled by default.** See [PRIVACY.md](PRIVACY.md) for what is and isn't collected.

- `/cre-skills:usage-stats` -- view your aggregated skill usage patterns
- `/cre-skills:feedback-summary` -- view your session ratings and comments
- Data stays in `~/.cre-skills/` (never leaves your machine)

v2.0.0 migrates from MIT to Apache 2.0. This provides attribution protection for contributors. See [NOTICE](NOTICE) for attribution details.

## Installation

### Upgrading from v1.0.0

```bash
cd /path/to/cre-skills-plugin
./scripts/update.sh
```

The update script preserves your `~/.cre-skills/` data (config, telemetry, feedback, brand guidelines).

### Fresh Install

```bash
# Option 1: Claude Code CLI (recommended)
claude plugin add github:mariourquia/cre-skills-plugin

# Option 2: One-line install
curl -fsSL https://raw.githubusercontent.com/mariourquia/cre-skills-plugin/main/scripts/install.sh | bash

# Option 3: Manual
git clone https://github.com/mariourquia/cre-skills-plugin.git
claude plugin add ./cre-skills-plugin

# Option 4: macOS DMG (available in v2.0.1 patch release)
# cre-skills-v2.0.0.dmg was not rebuilt for this release
```

### Verify Installation

```bash
./scripts/verify-install.sh
```
