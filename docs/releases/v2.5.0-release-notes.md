## What's New

### Construction Cost Estimator

A full-lifecycle construction cost estimation skill producing institutional-quality TDC budgets from concept through bid-level detail.

**11 workflows** covering:
- CSI MasterFormat division-level cost buildup (20 divisions)
- Regional cost adjustment with 60+ US city multipliers
- Site-specific adjustments (soil, demo, environmental, accessibility)
- Soft cost layering (A/E, permits, legal, financing, insurance, developer fee)
- Contingency framework calibrated by design stage
- Sensitivity analysis (labor, materials, timeline toggles)
- Comparable project benchmarking

**Visual capabilities**:
- **Floor plan input**: provide architectural drawings, sketches, or photos and get a cost estimate. The skill extracts dimensions, unit counts, structural systems, and finish materials from images.
- **Generative output**: describe a project concept and get ASCII floor plans and building massing diagrams with iterative refinement.
- **Co-creation loop**: iterate on design options with real-time cost feedback. Compare 3+ options side-by-side with full TDC breakdowns.

**New calculator**: `construction_estimator.py` -- zero-dependency Python CLI for automated cost estimation with JSON input/output.

**3 reference files**: 925-line CSI cost database, 60+ city regional factors, soft cost benchmarks with worked examples.

### Property Management Orchestrator

A deep command center that serves as the single entry point for all property management workflows. Routes to specialized downstream skills and adapts to any asset type.

**10 workflows**:
1. PM Triage Router (keyword-to-workflow matching)
2. Property Health Dashboard (aggregate KPIs with alert thresholds)
3. Monthly PM Report Generator (institutional-quality template)
4. Vendor Management Hub (contracts, scorecards, insurance, bids)
5. Tenant Relations Coordinator (communications, complaints, surveys)
6. Maintenance Operations Center (PM programs, work orders, emergency response)
7. Financial Operations (rent collection, delinquency, CAM, opex)
8. Compliance & Risk Monitor (inspections, permits, regulatory)
9. Capital Planning Coordinator (capex prioritization, reserves)
10. PM Performance Benchmarking (IREM/BOMA comparisons)

**8 asset-type module stubs** with frameworks for multifamily, office, retail, industrial, mixed-use, medical, self-storage, and hospitality. Each defines type-specific input fields, workflow outlines, key metrics, and integration points. Full deep-dive implementation targeted for v3.0.

**4 reference files**: IREM/BOMA KPI benchmarks, monthly report template, vendor management framework, asset-type module index.

### Space Planning & Redesign Orchestrator (Stub)

A multi-agent orchestrator for property managers evaluating space redesigns, repositioning strategies, and amenity additions. Defines 5 specialist subagents:

1. **Space Programmer** -- analyzes existing layouts, proposes program options
2. **Design Visualizer** -- generates floor plans and concept renderings
3. **Cost Estimator** -- delegates to construction-cost-estimator for pricing
4. **Market Validator** -- competitive audit, tenant demand surveys, rent premium estimation
5. **ROI Analyst** -- NPV/IRR analysis, 3-scenario modeling, investment recommendation

**4 reference files**: amenity cost benchmarks (16 types), tenant survey templates (3 instruments), competitive amenity audit framework, ROI model template with worked example.

Status: stub framework with full scope definition. Multi-agent dispatch targeted for v3.0.

## Installation

### New Install

**macOS**: Download `cre-skills-v2.5.0.dmg` from the [release page](https://github.com/mariourquia/cre-skills-plugin/releases/tag/v2.5.0).

**Windows**: Download `cre-skills-v2.5.0-setup.exe` from the [release page](https://github.com/mariourquia/cre-skills-plugin/releases/tag/v2.5.0).

**CLI**: `curl -fsSL https://raw.githubusercontent.com/mariourquia/cre-skills-plugin/main/scripts/install.sh | bash`

### Upgrade from v2.0.0

```bash
cd /path/to/cre-skills-plugin
./scripts/update.sh
```

Or pull manually:

```bash
git pull --ff-only origin main
```
