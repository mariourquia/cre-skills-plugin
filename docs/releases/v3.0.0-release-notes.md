# v3.0.0 — Feedback system

## Summary
Introduced the structured feedback system: `/cre-skills:send-feedback` and `/cre-skills:report-problem` commands, remote submission to a Vercel/Supabase backend, and the privacy-preserving outbox pattern.

## Highlights
- `/cre-skills:send-feedback` slash command for structured feedback
- `/cre-skills:report-problem` slash command for bug reports
- Remote submission via Vercel Function + Supabase backend
- `feedback-outbox.mjs` retry pattern for failed remote submissions
- PRIVACY.md documents the full feedback contract
