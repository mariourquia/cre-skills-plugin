# Lease Abstract Template

---

## TEMPLATE INSTRUCTIONS (strip before use)

This template produces a complete lease abstract for a single commercial lease. Abstracts are used by asset managers, acquisition teams, lenders, and property managers to quickly reference key lease terms without reading the full document.

**Accuracy standard**: Every field must be traceable to a specific section of the lease or amendment. If a field cannot be confirmed, mark it "[UNCONFIRMED -- see Note]" with a footnote describing the ambiguity. An abstract that states a term incorrectly is worse than one that flags an uncertainty.

**Completeness**: All 35 fields must be populated. If a field is not applicable to this lease type, enter "N/A" with a brief reason (e.g., "N/A -- office lease, no percentage rent").

**Update protocol**: When any amendment is executed, update the abstract and note the amendment date and affected fields in the Change Log at the bottom.
