# ROI Model Template

Standardized ROI calculation methodology for space planning and amenity investments. Designed for the ROI Analyst subagent (Workflow 5).

---

## Part 1: Investment Cost Framework

All costs flow into a single total project cost used as the denominator for return calculations.

### Cost Components

| Category | Line Items | Typical % of Total |
|---|---|---|
| **Hard construction** | Demo, framing, MEP, finishes, specialties | 40-55% |
| **FF&E** | Furniture, fixtures, equipment, artwork | 15-35% |
| **Technology** | AV, access control, WiFi, digital signage, booking systems | 5-15% |
| **Soft costs** | Architecture, engineering, permits, project management | 12-20% |
| **Contingency** | Unforeseen conditions, scope changes, price escalation | 10-15% |

### Cost Build-Up Template

| Line Item | Amount | Source |
|---|---|---|
| Hard construction (see amenity-cost-benchmarks.yaml) | $ | Estimate / Bid / Benchmark |
| FF&E procurement | $ | Spec / Quote / % of hard cost |
| Technology systems | $ | Spec / Quote / $/SF benchmark |
| Architecture & engineering | $ | Fee proposal / % of hard cost |
| Permits and fees | $ | Jurisdiction schedule |
| Project management | $ | Internal / 3rd party fee |
| Testing and inspections | $ | Code requirements |
| Move/temporary relocation | $ | If displacing existing tenants |
| Contingency (% of above) | $ | 10% for renovation, 15% for complex |
| **Total project cost** | **$** | |

## Part 2: Revenue Sources

Amenity investments generate value through four channels. Not all apply to every project.

### Channel 1: Rent Premium

The incremental rent achievable because the property now has the amenity.

| Input | Value | Source |
|---|---|---|
| Building rentable SF or unit count | | Rent roll |
| Current asking rent ($/SF/yr or $/unit/mo) | | Leasing team |
| Estimated premium ($/SF/yr or $/unit/mo) | | Competitive audit, broker opinion |
| % of building affected by premium | | Realistic capture rate (not 100%) |
| Phase-in period (months to full capture) | | Typically 12-24 months at lease renewal |
| **Annual incremental rent at stabilization** | **$** | Premium x affected SF x capture rate |

### Channel 2: Direct Revenue

Revenue generated by the amenity itself (coworking fees, event rentals, parking premiums).

| Input | Value | Source |
|---|---|---|
| Revenue per unit/hour/month | | Market rate / competitive set |
| Projected utilization rate | | Demand survey, industry benchmarks |
| Operating months per year | | 12 unless seasonal |
| Less: operating expenses | | Staffing, maintenance, supplies |
| **Annual net direct revenue** | **$** | Gross revenue - operating expenses |

### Channel 3: Retention Value

The avoided cost of tenant turnover attributable to the amenity.

| Input | Value | Source |
|---|---|---|
| Annual tenant turnover rate (current) | | Property records |
| Estimated turnover reduction | | Survey Q7 results, industry data |
| Average turnover cost per tenant/unit | | Vacancy loss + TI + LC + make-ready |
| Number of tenants/units | | Rent roll |
| **Annual retention value** | **$** | Turnover reduction x cost per turnover x units |

*Typical turnover costs: Office = 6-12 months rent equivalent. Multifamily = 1-3 months rent + $2K-$8K make-ready.*

### Channel 4: Lease-Up Acceleration

Faster absorption of vacant space because the amenity makes the property more competitive.

| Input | Value | Source |
|---|---|---|
| Current vacant SF or units | | Rent roll |
| Current absorption rate (SF or units/month) | | Leasing pipeline |
| Projected absorption rate post-amenity | | Broker estimate, comp analysis |
| Months of accelerated lease-up | | Delta in time to stabilize |
| Rent on accelerated SF/units | | Asking rent |
| **One-time lease-up value** | **$** | Months saved x (vacant units x monthly rent) |

## Part 3: Three-Scenario Model

Every amenity investment must be modeled across three scenarios to bracket uncertainty.

### Scenario Assumptions

| Variable | Bear Case | Base Case | Bull Case |
|---|---|---|---|
| Rent premium capture | 50% of estimate | 75% of estimate | 100% of estimate |
| Occupancy improvement | 0% | 2-3% | 5% |
| Direct revenue utilization | 40% | 60% | 80% |
| Turnover reduction | 0% | 50% of estimate | 100% of estimate |
| Construction cost overrun | +15% | +5% | 0% |
| Phase-in period | 24 months | 18 months | 12 months |

### Scenario Output Template

| Metric | Bear | Base | Bull |
|---|---|---|---|
| Total project cost | $ | $ | $ |
| Annual incremental NOI (stabilized) | $ | $ | $ |
| Simple payback (years) | | | |
| NPV (5-year, at discount rate) | $ | $ | $ |
| IRR (5-year) | % | % | % |
| Property value creation | $ | $ | $ |

## Part 4: NPV Calculation

### Formula

```
NPV = -C0 + SUM[ CFt / (1 + r)^t ] for t = 1 to T
```

Where:
- **C0** = Total project cost (from Part 1)
- **CFt** = Net incremental cash flow in year t (rent premium + direct revenue + retention value - incremental operating costs)
- **r** = Discount rate
- **T** = Analysis period (typically 5 years, or remaining hold period)

### Discount Rate Selection

| Approach | Rate | When to Use |
|---|---|---|
| Property cap rate | Typically 5-8% | Default for single-asset analysis |
| WACC | Varies by capital structure | When fund/portfolio-level hurdle applies |
| Opportunity cost | IRR of next-best capital deployment | When comparing competing capex projects |
| Risk-adjusted | Cap rate + risk premium (1-3%) | When project has above-average execution risk |

**Guideline**: Use the property's going-in cap rate as the base discount rate. Add 1-2% for projects with construction risk, tenant disruption, or demand uncertainty.

## Part 5: IRR Calculation

### Cash Flow Schedule

| Year | Investment | Rent Premium | Direct Revenue | Retention Value | Operating Cost | Net Cash Flow |
|---|---|---|---|---|---|---|
| 0 | (Total cost) | -- | -- | -- | -- | (Total cost) |
| 1 | -- | Phase-in | Phase-in | Phase-in | Full year | |
| 2 | -- | Phase-in | Stabilized | Stabilized | Full year | |
| 3 | -- | Stabilized | Stabilized | Stabilized | Full year | |
| 4 | -- | Stabilized | Stabilized | Stabilized | Full year | |
| 5 | -- | Stabilized | Stabilized | Stabilized | Full year | |

IRR is the rate **r** that sets NPV = 0 across this cash flow schedule.

### Reinvestment Assumption

IRR assumes intermediate cash flows are reinvested at the IRR itself. For projects with IRR significantly above the discount rate, also calculate **Modified IRR (MIRR)** using the discount rate as the reinvestment rate for a more conservative view.

---

## Part 6: Simple Payback Period

### Formula

```
Payback = Total Project Cost / Annual Incremental NOI (stabilized)
```

**Adjustment for phase-in**: If the amenity takes 12-18 months to reach stabilized revenue, calculate actual payback by summing yearly cash flows until cumulative cash flow turns positive.

### Benchmarks

| Payback Period | Signal |
|---|---|
| < 3 years | Strong investment. Proceed with confidence. |
| 3-5 years | Acceptable for most hold periods. Standard amenity investment range. |
| 5-7 years | Marginal. Requires strong strategic rationale (competitive necessity, retention crisis). |
| > 7 years | Weak financial case. Only justified by regulatory requirement or existential competitive threat. |

---

## Part 7: Property Valuation Impact

### Formula

```
Value Creation = Annual Incremental NOI / Cap Rate
```

This is the most powerful metric for ownership. It translates operating improvements into asset value.

### Template

| Input | Value |
|---|---|
| Annual incremental NOI (stabilized, base case) | $ |
| Property cap rate | % |
| **Implied value creation** | **$** |
| Total project cost | $ |
| **Value creation multiple** | **X** (value created / cost invested) |

**Target**: Value creation multiple > 2.0x means every dollar invested creates two or more dollars of property value.

## Part 8: Worked Example

### Scenario: Lobby Renovation

**Property**: 50,000 SF Class B office building, 82% occupied, asking $28/SF NNN.

**Investment**:

| Line Item | Amount |
|---|---|
| Hard construction (2,500 SF lobby at $120/SF) | $300,000 |
| FF&E (25% of hard cost) | $75,000 |
| Technology (digital directory, access, AV) | $25,000 |
| Soft costs (15% of hard cost) | $45,000 |
| Contingency (10%) | $44,500 |
| **Total project cost** | **$489,500** |

**Revenue projection (base case)**:

| Source | Annual Value |
|---|---|
| Rent premium: $2.00/SF on 50,000 SF at 75% capture | $75,000 |
| Occupancy improvement: 3% on 50,000 SF at $28/SF | $42,000 |
| Retention value: 1 avoided turnover at $50K cost | $50,000 |
| Less: incremental operating cost ($6/SF on 2,500 SF) | ($15,000) |
| **Annual incremental NOI** | **$152,000** |

**Returns (base case)**:

| Metric | Value |
|---|---|
| Simple payback | 3.2 years |
| NPV (5 years at 7% discount rate) | $133,700 |
| IRR (5 years) | 17.4% |
| Value creation ($152K / 7.0% cap) | $2,171,000 |
| Value creation multiple | 4.4x |

**Verdict**: BUILD. IRR of 17.4% exceeds cost of capital. Payback under 5 years. Positive NPV. Value creation multiple of 4.4x is strong. Even the bear case (50% of revenue, +15% cost) produces a 5.8-year payback and positive NPV.

## Part 9: Decision Framework

### Decision Criteria

An amenity investment passes the decision gate if ALL of the following are true in the base case:

| Criterion | Threshold | Rationale |
|---|---|---|
| IRR > cost of capital | Varies (typically 7-12%) | Project must exceed the hurdle rate |
| Simple payback < 5 years | 5 years | Must recover investment within typical hold period |
| NPV > 0 | Positive | Project creates value net of time value of money |
| Value creation multiple > 1.5x | 1.5x | Each dollar invested creates $1.50+ in property value |
| Bear case NPV > -50% of investment | Limited downside | Downside scenario does not destroy significant capital |

### Decision Matrix

| Base Case IRR | Payback | NPV | Recommendation |
|---|---|---|---|
| > hurdle + 5% | < 3 years | Strongly positive | **BUILD** -- high confidence |
| > hurdle | 3-5 years | Positive | **BUILD** -- standard approval |
| Near hurdle | 5-7 years | Near zero | **DEFER** -- improve assumptions or wait for better timing |
| < hurdle | > 7 years | Negative | **PASS** -- redeploy capital elsewhere |

### Alternative Capital Deployment Check

Before recommending BUILD, compare the proposed amenity investment to:

1. **Unit renovations**: Would the same capital generate higher returns if deployed to kitchen/bath upgrades? (Typical IRR: 15-25% for multifamily unit reno.)
2. **Leasing incentives**: Would spending the capital on free rent or TI allowances produce faster lease-up? (Compare NPV of concession vs. amenity.)
3. **Debt paydown**: Does reducing leverage improve DSCR covenant headroom or enable a refi at better terms?
4. **Acquisition**: Could the capital fund a down payment on an additional asset with higher risk-adjusted returns?

If the amenity investment has the highest risk-adjusted return among alternatives, the BUILD recommendation stands. If not, note the superior alternative in the investment memo.
